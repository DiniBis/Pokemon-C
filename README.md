# Pokemon like en C
Projet scolaire pour pratiquer C

## Fonctionnalités:
- Affichage du shop, avec quantité possédée et argent
- Structure:
  - du joueur :
    - son nom
    - son pokemon actif
    - son argent
  - d'objets :
    - son nom
    - la quantité possédée
    - son prix
  - de Pokemon :
    - ses stats
    - son niveau
    - son EXP
    - ses capacités
  - de capacité :
    - son impact sur les stats
    - ses dégâts
- thread pour la sauvegarde automatique
